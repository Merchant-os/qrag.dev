================================================================================
                          Quick Start Guide
================================================================================

1. Install Dependencies (Linux/macOS):
   $ bash setup.sh

   Or (Windows):
   > setup.bat

2. Set OpenAI API Key (required for embeddings):
   $ export OPENAI_API_KEY="your-openai-key"

   GLM key is already embedded, but you can override:
   $ export GLM_API_KEY="your-glm-key"  # Optional

3. Generate Real Embeddings:
   $ python rag/embed.py

4. Generate Q# Code with RAG:
   $ python -c "import sys; sys.path.insert(0, '.'); from agent.agent import generate; print(generate('Create a Bell state', top_k=5))"

================================================================================
                         Usage Examples
================================================================================

Basic RAG Generation:
------------------
from agent.agent import generate

Baseline Generation (No Retrieval):

Direct Retrieval:
------------------
from rag.retrieve import retrieve

results = retrieve("Grover search algorithm", top_k=3)
for r in results:
    print(f"[{r['similarity']}] {r['id']}")
