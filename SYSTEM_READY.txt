HOW EVERYTHING WORKS TOGETHER:

1. You ask: "Create a Bell state"

2. RAG Agent:
   → Retrieves 3 similar examples from corpus
   → Passes them to GLM 4.7
   → Generates accurate Q# code

3. Baseline Agent:
   → Passes nothing to GLM 4.7
   → Generates Q# code from knowledge

4. Both use:
   → Same GLM 4.7 API key
   → Same QDK 1.0+ syntax rules
   → Same system prompt

# Using in your own scripts
import sys
sys.path.insert(0, '.')

from agent.agent import generate
from rag.retrieve import retrieve

def generate_with_rag(task: str, top_k: int = 5) -> str:
    examples = retrieve(task, top_k=top_k)
    print(f"Retrieved {len(examples)} examples...")
    return generate(task, top_k=top_k)

# Your custom function
code = generate_with_rag("Implement Grover's search", top_k=3)
print(code)
