═══════════════════════════════════════════════════════════════════════════════════════

                         WHAT EACH COMPONENT DOES
═══════════════════════════════════════════════════════════════════════════════════════════

RAG Agent (agent/agent.py):
  • Retrieves relevant Q# examples
  • Passes them to GLM 4.7
  • Generates accurate Q# code
  • Use: generate(task, top_k=5)

Baseline Agent (agent/baseline.py):
  • No retrieval
  • Pure GLM 4.7 generation
  • For comparison
  • Use: generate(task)

Retrieval (rag/retrieve.py):
  • Semantic search using embeddings
  • Cosine similarity
  • Returns top-k matches
  • Use: retrieve(query, top_k=3)
