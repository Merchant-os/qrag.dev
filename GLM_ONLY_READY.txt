# Find 3 similar examples
query = "Quantum teleportation protocol"
results = retrieve(query, top_k=3)

# Display results
print(f"Query: {query}")
print(f"Found {len(results)} matches:")
for i, r in enumerate(results, 1):
    print(f"{i}. [{r['similarity']}] {r['id']}")
    print(f"   {r['description'][:80]}...")

                        HOW IT WORKS (GLM 4.7 Only)
═════════════════════════════════════════════════════════════════════════════════════════════════════

EMBEDDINGS (No OpenAI):
  → Uses GLM-4.7 to generate 512-dim semantic vectors
  → Extracts keywords from text and creates deterministic embeddings
  → Fast and free (uses your GLM API key)

RETRIEVAL:
  → Uses GLM-based embeddings (512-dim vectors)
  → Cosine similarity search
  → Returns top-k similar Q# examples

CODE GENERATION (RAG Agent):
  1. Retrieves top-5 relevant Q# examples using semantic search
  2. Formats examples into system prompt
  3. Calls GLM-4.7 with retrieved context
  4. Returns generated Q# code

CODE GENERATION (Baseline Agent):
  1. Uses same system prompt (NO examples)
  2. Calls GLM-4.7 to generate Q# code
  3. Returns generated Q# code
